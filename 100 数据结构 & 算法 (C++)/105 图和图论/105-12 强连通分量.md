# 1. 强连通和强连通分量
强连通：有向图G种的任意两个结点均联通。
强连通分量：极大的强连通子图。
# 2. DFS生成树
我们使用DFS算法搜索一张有向图，显然每一个点均有其被搜索到的对应顺序，形似一颗树。
![[Pasted image 20260612192145.png]]
以上面这颗类似于前序遍历的DFS生成树为例，主要有三种边构成：

1. 树边：当每次搜索找到一个还没有被搜索过的结点的时候就形成了一条边；
2. 返祖边（回边）：搜索到该结点的祖宗结点时形成的边（图中 7→ 1 ），即指向祖先结点的边；
3. 横叉边：搜索到一个被访问过的非祖宗结点时形成的边（图中 9→7 ）。

其实还要一种前向边 （图中 3→6），是搜索的时候遇到子树中的结点形成的边，不过接下来暂不讨论这类边。
# 3. Tarjan算法
Tarjan算法通过使用 DFS 来求强连通分量，在搜索的过程中我们维护一个栈，将搜索树中被搜索到的未处理的节点加入栈中。

我们为每个节点维护两个值：

1. $dfn_u$ ：DFS 中点 u 被搜索到的次序
2. $low_u$ ： 点 u 的子树能够回溯到的最早的已经在栈中的节点。其定义为：以 u 为根的 DFS 子树中的节点，通过**一条不在 DFS 树上的边**能够到达的所有节点中， $dfn$ 的最小值

接下来我们开始 DFS ，在过程中，我们将路过的每一个点都塞进一个栈里，当我们从点 u 搜索下一个点 v 的时候，考虑下面几种情况：

1. 如果点 v 没有被搜索过（对应普通的树边），我们就搜下去。在搜完下面所有的子树的时候，会发生回溯，此时如果 v 能回溯到更早的地方，那 u 对应地也能回溯到达，此时 $low_u =\min(low_u,low_v)$ ，接着我们继续递归搜索
2. 如果点 v 被搜索过并且在栈里（对应返祖边/横叉边），此时说明出现了一个环，此时从 u 开始回溯是能回溯到 v 的，并且显然 v 是这个环的 dfn 最小的那个点，所以我们回溯就能将包括 u 在内的所有环上的点都更新为 v 的 low 值， $low_u = \min(low[u],dfn[v])$ ，此时 v 的 dfn = low ，我们从 u 开始，回溯并且将环上的每一个点进行弹栈处理，直到将 v 也弹出去为止，就说明我们已经处理完了一个强连通分量
3. 如果点 v 被搜索过并且不在栈里（对应横叉边），说明 v 已经被处理了，即 v 在上一种情况中已经被弹栈了，属于另一个强连通分量，同时由于是有向边，v 无法指向 u，因此无法构成强连通分量，我们就不处理这条边，忽略即可。

```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;

constexpr int N = 1e4+10;
int n,m,t,scc_cnt,top;
vector<int> g[N],lis[N];
int dfn[N],low[N],scc[N],scc_w[N],stk[N];
bool vis[N],pt[N];

void tarjan(int u) {
    dfn[u] = low[u] = ++t;
    stk[++top] = u;
    vis[u] = true;
    for (int v : g[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u],low[v]);
        }
        else if (vis[v]) low[u] = min(low[u],dfn[v]);
    }
    if (dfn[u] == low[u]) {
        scc_cnt++;
        while (true) {
            int v = stk[top--];
            vis[v] = false;
            scc[v] = scc_cnt;
            scc_w[scc_cnt]++;
            if (u == v) break;
        }
    }
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    cin >> n >> m;
    for (int i = 1 ; i <= m ; i++) {
        int u,v; cin >> u >> v;
        g[u].push_back(v);
    }
    for (int i = 1 ; i <= n ; i++) if (!dfn[i]) tarjan(i);
    for (int i = 1 ; i <= n ; i++) lis[scc[i]].push_back(i);
    for (int i = 1 ; i <= scc_cnt ; i++) sort(lis[i].begin(),lis[i].end());
    cout << scc_cnt << '\n';
    for (int i = 1 ; i <= n ; i++) {
        if (pt[scc[i]]) continue;
        for (auto u : lis[scc[i]]) {
            cout << u << ' ';
        }
        cout << '\n';
        pt[scc[i]] = true;
    }
}
```
