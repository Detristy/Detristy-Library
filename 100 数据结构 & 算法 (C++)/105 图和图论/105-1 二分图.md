# 1. 二分图
二分图是一类特殊的图，其顶点集可以被划分为两个互不相交的子集，使得图中的每条边都连接这两个集合之间的一对点。

严格的定义如下：
>[!definition] 定义 1（二分图）
>如果图 $G=(V,E)$ 的顶点集 $V$ 可以分为两个互不相交的子集 $X$ 和 $Y$ ，使得每条边 $e\in E$ 的两个端点都分别属于 $X$ 和 $Y$ ，就称图 $G$ 是一个 **二分图**（bipartite graph）．集合 $X$ 和 $Y$ 常称作它的两个 **部分**（part），或者分别称为二分图的左部和右部．当二分图的两个部分 $X$ 和 $Y$ 已知时，也可以用三元组 $(X,Y,E)$ 来表示二分图 $G$ 。

# 2. 二分图染色问题

对于判断一个图是否是二分图，我们可以通过染色的方法解决，对于二分图，我们只需要染两种颜色即可。

确定一个点作为我们染色的起点，接着从这个点开始bfs遍历这张图，我们为相邻的点染上不同的颜色。

显然，对于每一个点我们有三种可能的状态，没染色，染了颜色 A 和染了颜色 B 。

我们怎么控制不同的颜色呢？很简单，假如我们为染了颜色A赋值为 1，染颜色 B 赋值为 2，那么我只需要用 3 减去当前颜色的赋值就可以得到另一种颜色的赋值。

```cpp
for (int i = 1 ; i <= n ; i++) {  
    if (color[i] == 0) {  
        int cnt1 = 0,cnt2 = 0;  
        queue<int> q;  
        color[i] = 1;  
        cnt1++;  
        q.push(i);  
        while (!q.empty()) {  
            int u = q.front();  
            q.pop();  
            for (auto v : g[u]) {  
                if (color[v] == 0) {  
                    color[v] = 3 - color[u];  
                    if (color[v] == 1) cnt1++;  
                    else cnt2++;  
                    q.push(v);  
                }  
                else if (color[v] == color[u]) {  
                    cout << "Impossible" << endl;  
                    return 0;  
                }  
            }  
        }  
        tot += min(cnt1,cnt2);  
    }  
}
```

# 3. 匈牙利算法

匈牙利算法用于解决二分图最大匹配的问题，本质上是通过借位构造出一个最长的**增广路**。

#### 图的匹配
设 $G=(V,E)$ 是一张无向图。如果一组边 $M\subseteq E$ 不包含自环，且两两之间没有公共顶点，那么，边集 $M$ 就称为图 $G$ 的一个 **匹配** 或 **独立边集**，其中边数最多的匹配为**最大匹配**。
#### 增广路
**增广路**是始于未匹配点且终于未匹配点的**交错路**。
#### Berge引理
>[!lemma] 引理 1（Berge引理）
> 图 $G$ 中的一个匹配 $M$ 是最大匹配，当且仅当 $G$ 中不存在关于 $M$ 的增广路。

这个引理我们在这里暂时不证。
由Berge引理，当找不到增广路时，就说明已经得到了最大匹配
### 匈牙利算法
匈牙利算法的大致过程如下：

1. A 尝试位置 P，如果 P 目前为空，就让 A 占用 P。
2. 如果 P 被占用了，假设目前 B 占用 P，则试着让 B 搜索新的位置。如果 B 搜索到了一个新的空位置 Q ，就让 B 占用 Q，然后就可以让 A 占用 P。

此时 A-P-B-Q 就是一条交错路
板子如下，ans是总匹配数：

```cpp
#include <iostream>
#include <vector>
#include <cstring>

using namespace std;

const int MAXN = 1005; 
int n, m, e;
vector<int> adj[MAXN]; 
int match[MAXN];     
bool vis[MAXN];      

bool dfs(int u) {
    for (int v : adj[u]) {
        if (!vis[v]) {
            vis[v] = true;
            if (match[v] == 0 || dfs(match[v])) {
                match[v] = u; 
                return true;
            }
        }
    }
    return false; 
}

int main() {
    cin >> n >> m >> e;
    for (int i = 0; i < e; i++) {
        int u, v;
        cin >> u >> v;
        if (v > m) continue; 
        adj[u].push_back(v);
    }
    int ans = 0;
    for (int i = 1; i <= n; i++) {
        memset(vis, 0, sizeof(vis));
        if (dfs(i)) ans++;
    }
    cout << ans << endl;
    return 0;
}
```
