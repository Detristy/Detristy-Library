# 1. 拓扑序

>[!definition] 定义 1 （拓扑序）
> 对于一个有向图 $G=<V,E>$ ，一个顶点集 $V$ 上的全序关系 $\leq_T$ 被称为拓扑序，当且仅当对于每一条边 $<u,v>\in E$ ，在序列中满足 $u\leq_T v$

# 2. 拓扑排序

拓扑排序 (Topological Sort) 就是在一个有先后关系的图（AOV网）里找出一个合法的顺序。

我们明确，既然是一个有先后关系的图，那么这个图必然没有环，只有没有环的图才能进行拓扑排序。

对于拓扑排序，我们考虑点的入度，我们可以将入度比作前置任务：
假如入度为 $0$ ，那么没有前置任务，现在可以开始执行；
入度 $>0$ ，那么还有前置任务，我们应等待前置任务完成。

不难发现，任务服从先进先出的顺序关系，于是我们可以想到使用队列来维护。

我们每一次都需要从队头取出一个任务完成，此时假设 $v$ 的前置任务是 $u$ 和 $m$，此时我们完成了 $u$ ，那么 $v$ 的前置任务数量（入度）减 $1$。
如果我们在某次任务执行完成了 $m$ ，此时 $v$ 的任务数量就为 0，此时我们就可以执行 $v$ 了，将 $v$ 塞入队列中等待被执行。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int n;
const int N = 102;
vector<int> g[N];
queue<int> res;
int rd[N];
queue<int> q;

int main() {
    cin >> n;
    for (int i = 1 ; i <= n ; i++) {
        int x;
        while (cin >> x && x) {
            g[i].push_back(x);
            rd[x]++;
        }
    }
    for (int i = 1 ; i <= n ; i++){
        if (rd[i] == 0) q.push(i);
    }
    while (!q.empty()) {
        int x = q.front();
        q.pop();
        for (auto i : g[x]) {
            rd[i]--;
            if (rd[i] == 0) q.push(i);
        }
        res.push(x);
    }
    for (int i = 1 ; i <= n ; i++) {
        cout << res.front() << " ";
        res.pop();
    }
    return 0;
}
```

# 3. 关键路径
关键路径是定义在 AOE网 上的，AOE网指的是用顶点表示事件（Event），有向边表示活动（Activity），边的权值表示活动持续时间的有向无环图。

关键路径是从源点到汇点的最长路径，路径长度为路径上各边边权和，最长路径决定了最短的工期。

关键活动是最早开始时间 $e$ 等于最晚开始时间 $l$ 的活动，关键路径由关键活动组成。

要求解关键路径，我们用拓扑排序，求出每一个活动最早的发生时间，再逆向拓扑排序求出活动的最晚发生时间，接着求解两者的差，差值为 $0$ 的活动就是关键活动。

```cpp
#include <stdio.h>
#include <string.h>

#define MAXV 1005

typedef struct {
    int to, w, next;
} Edge;

Edge edges[MAXV * MAXV];
int head[MAXV], cnt;
int inDeg[MAXV];
int ve[MAXV], vl[MAXV];       // 事件最早/最晚发生时间
int topoOrder[MAXV], topoCnt; // 拓扑序列
int n, m;                      // n个事件, m个活动

void addEdge(int u, int v, int w) {
    edges[cnt].to = v;
    edges[cnt].w = w;
    edges[cnt].next = head[u];
    head[u] = cnt++;
    inDeg[v]++;
}

// 拓扑排序 + 求 ve[]
int topoSort() {
    int queue[MAXV], front = 0, rear = 0;
    topoCnt = 0;
    memset(ve, 0, sizeof(ve));

    for (int i = 0; i < n; i++)
        if (inDeg[i] == 0) queue[rear++] = i;

    while (front < rear) {
        int u = queue[front++];
        topoOrder[topoCnt++] = u;  // 记录拓扑序
        for (int i = head[u]; i != -1; i = edges[i].next) {
            int v = edges[i].to, w = edges[i].w;
            if (ve[u] + w > ve[v])
                ve[v] = ve[u] + w;  // 取最大值
            if (--inDeg[v] == 0)
                queue[rear++] = v;
        }
    }
    return topoCnt == n;  // 判断是否有环
}

// 求关键路径
void criticalPath() {
    if (!topoSort()) return;  // 有环则无关键路径

    // 初始化 vl[] 为汇点的 ve 值
    int maxVe = 0;
    for (int i = 0; i < n; i++)
        if (ve[i] > maxVe) maxVe = ve[i];
    for (int i = 0; i < n; i++)
        vl[i] = maxVe;

    // 逆拓扑序求 vl[]
    for (int idx = topoCnt - 1; idx >= 0; idx--) {
        int u = topoOrder[idx];
        for (int i = head[u]; i != -1; i = edges[i].next) {
            int v = edges[i].to, w = edges[i].w;
            if (vl[v] - w < vl[u])
                vl[u] = vl[v] - w;  // 取最小值
        }
    }

    // 遍历每条活动，判断是否为关键活动
    for (int u = 0; u < n; u++) {
        for (int i = head[u]; i != -1; i = edges[i].next) {
            int v = edges[i].to, w = edges[i].w;
            int e = ve[u];        // 活动最早开始时间
            int l = vl[v] - w;    // 活动最晚开始时间
            if (e == l) {
                // <u, v> 是关键活动
            }
        }
    }
}
```
