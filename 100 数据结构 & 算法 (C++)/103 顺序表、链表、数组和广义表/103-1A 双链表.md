双链表对[[103-1 单链表 | 单链表]] 进行了优化。

# 1. 双链表的构建

## 1.2 双链表的特点

双链表通过引入一个新的指针 `prev` 能够实现 $O(1)$ 访问前驱结点，同时使得链表能够进行双向遍历。属于一种空间换时间的方法。

![[Pasted image 20260521201038.png]]

## 1.1 结点

和单链表不同之处在于多了一个前驱指针。

```cpp
struct Node {  
    int num = 0,score = 0;  
    Node *next, *prev;  
    Node() {  
        next = NULL;  
        prev = NULL;  
    }  
};
```

注意保证头节点的 prev 和尾节点的 next 为空指针。

# 2. 双链表操作

## 2.1 插入

这里就不区分尾插和头插了，原理是一样的。
因为引入了一个指针，所以更要注意修改顺序。

要在 p 后插入一个结点 s，分为四步：
1. 新节点 next 指向 p 的后继；
2. 原来的 p 的后继的 prev 指向 s；
3. s 的 prev 指向 p；
4. p 的 next 指向 s。

前两部必须在第四步之前完成否则会导致后链丢失。

```cpp
bool InsertNextDNode(Node *p, Node *s) {
    if (p == NULL || s == NULL) return false;
    s->next = p->next;
    if (p->next != NULL)
        p->next->prev = s;
    s->prev = p;
    p->next = s;
    return true;
}
```

## 2.2 删除

因为有 prev 指针的存在，所以删除就方便很多。

假设删除 p 的后继结点 q，分为三步：
1. p 的 next 指向 q 的后继；
2. 如果 q 存在后继，那么 q 的后继的 prev 指向 p；
3. 删除 q 结点。

```cpp
bool DeleteNextDNode(Node *p) {
    if (p == NULL || p->next == NULL) return false;
    Node *q = p->next;
    p->next = q->next;
    if (q->next != NULL)
        q->next->prev = p;
    free(q); //或者 delete p
    return true;
}
```
