# 1. 概念

## 1.1 广义表的定义
 广义表是 $n\;(n\geq 0)$ 个元素组成的有限序列，记作：

$$
LS = (a_{1},a_{2},\dots,a_{n})
$$

 其中 $a_i$ 可以是不可再分的原子，也可以是另一个广义表（子表）。

 广义表是线性表的推广，广义表的子表本身也是广义表（广义表是被递归定义的），广义表可被多个表共享。

注意需要特别区分：$()$ 是空表（长度 0），而 $(())$ 是含空表的非空表（长度 1）。

## 1.2 表头与表尾
对于非空广义表，表头是第一个元素 $a_1$，可为原子或子表；而表尾是除表头外其余元素组成的广义表，**表尾必定是一个子表**。

| $LS$              | 表头           | 表尾         |
| ---- | ------------ | ---------- |
| $(a, b)$          | $a$          | $(b)$      |
| $(x, (a, b))$     | $x$          | $((a, b))$ |
| $((x, (a,b)), y)$ | $(x, (a,b))$ | $(y)$      |

>[!warning]
> 因为表尾必定是一个子表，因此表尾始终加括号：$(a,b)$ 的表尾是 $(b)$ 而非 $b$。

## 1.3 长度与深度

广义表的长度指的是最上层元素的个数，空表长度为 0 。
广义表的深度指的是括号的最大嵌套层数，空表为 1，原子为 0 。

对于一个表，里面存储的可能是原子或子表，因此我们需要用引入一个变量 tag （称为标志位）来区分，默认 tag = 0 时为原子，tag = 1 时为子表。

链表中的每个结点由三部分构成：标志位，数据域，指针域。其中指针域存放与本元素同层的下一个元素对应结点的地址。
