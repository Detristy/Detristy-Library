# 1. 概念
优先队列是一种根据优先级出队的队列，通过[[102-6 堆 | 堆]] 实现。

通常会使用封装好的 priority_queue 来实现。
# 2. 声明
#### 大根堆
优先队列默认使用大根堆来实现，声明方式如下：
```cpp
priority_queue<int> pq;
```
#### 小根堆
如果要使用小根堆，需要通过使用声明 `greater<>` 来实现，而这个声明是一个比较类，其模板参数是按照顺序继承的，其中这个声明是第三个参数，而第二个参数是承载数据的类型（默认是vector），因此我们要把第二个参数也显示声明出来。
声明方式如下：
```cpp
priority_queue<int,vector<int>,greater<int>> pq;
```
#### 自定义排序
当我们使用结构体作为优先队列的元素的时候，需要单独写结构体的排序。
```cpp
struct Node {
    int id, dist;
};

struct cmp {
    // 注意：优先队列的比较逻辑和 std::sort 是反着的！
    // 在 priority_queue 中，若返回 true，代表优先级低，会被排在后面
    bool operator()(const Node& a, const Node& b) {
        return a.dist > b.dist; // 距离越大的，优先级越低（即实现小根堆）
    }
};

// 声明一个针对 Node 的小根优先队列
priority_queue<Node, vector<Node>, cmp> pq;
```

# 3. 调用
优先队列提供了下列函数：
1. `push(x)` 将元素推入队列
2. `pop()` 弹出优先级最高的元素
3. `top()` 查看优先级最高的元素
4. `empty()` 判空

总体而言，维护优先队列的时间复杂度为 $O(\log n)$ 。
