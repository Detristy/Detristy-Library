# 1. 十字链表
考虑一张图的邻接表，每一个点引出一个链表，显然我们能通过扫描一次链表就得到这个点对应的出度，但如果要求这个节点的入度，则需要遍历整张邻接表来计数。
而如果考虑逆向建图，我们就能同理快速求出入度数量，但是此时出度数量我们还是需要遍历整张表。

此时我们引入十字链表，使得不管是查入度还是出度都能变得方便。
# 2. 十字链表对于图的构建
## 2.1 十字链表的结点
为了实现这个网络，我们需要重新设计结点。

这里注意区分链表结点和图的结点。

我们重新考虑一条边，对于一条有向边对应的链表结点 `arcNode` ，我们需要存储五个信息：起点 `tailvex` 、终点 `headvex` 、头向指针 `hlink` 、尾向指针 `tlink` 和边权。我们解释一下头向指针和尾向指针：

1. 头向指针（Header Link）：其指向下一个以相同终点为目标的边。
2. 尾向指针（Tail Link）：其指向下一个以相同起点出发的边。

显然，只要根据头向指针遍历一遍，我们就能找到所有指向同一个结点的所有点，从而求出入度。
同理，只要根据尾向指针遍历一遍，我们就能求出出度。

接下来我们考虑点，对于一个点，我们需要存储三个信息：点权、头向链表头和尾向链表头。其中：

1. 头向链表头 `first_in` ：指向第一条以当前结点为终点的边。
2. 尾向链表头 `first_out` ：指向第一条以当前结点为起点的边。

#### 有向边
```cpp
struct arcNode {  
    int tailvex,headvex;  
    arcNode * hlink, * tlink;  
    int w;  
    arcNode(int u,int v,int w):  
        tailvex(u),headvex(v),w(w),  
        hlink(NULL),tlink(NULL){}  
};
```

#### 图上结点
```cpp
struct vexNode {  
    string data;  
    arcNode * first_in, * first_out;  
    vexNode(string data):data(data),first_in(NULL),first_out(NULL){}  
};
```

## 2.2 十字链表的构造
#### 结点
对于所有结点，我们用一张顺序表来存储，称为顶点表 `vexList` 。
```cpp
vector<vexNode> vexList;
```
#### 有向边
对于每一条有向边 $u\to v$ ，我们记其为 `edge` ，那么这条边则要插入到 $u$ 的出度链表中（头插），要插入到 $v$ 的入度链表中（头插）。
 ```cpp
 void insert_edge(int u,int v,int w) {  
    arcNode * edge = new arcNode(u,v,w);  
    edge->tlink = vexList[u].first_out;  
    vexList[u].first_out = edge;  
    edge->hlink = vexList[v].first_in;  
    vexList[v].first_in = edge;  
}
 ```

## 2.3 十字链表的遍历
我们考虑遍历 $u$ 的所有出度和入度，只需要像普通的链表遍历一样即可。
#### 遍历出度
```cpp
void printOutEdge(int u) {  
    arcNode * p = vexList[u].first_out;  
    while (p != NULL) {  
        cout << vexList[p->headvex].data << " -> ";  
        p = p->tlink;  
    }  
}
```
#### 遍历出度
```cpp
void printInEdge(int u) {  
    arcNode * p = vexList[u].first_in;  
    while (p != NULL) {  
        cout << vexList[p->tailvex].data << " -> ";  
        p = p->hlink;  
    }  
}
```

## 2.5 完整结构
从而我们就实现了使用十字链表作为邻接表。

```cpp
struct arcNode {  
    int tailvex,headvex;  
    arcNode * hlink, * tlink;  
    int w;  
    arcNode(int u,int v,int w):  
        tailvex(u),headvex(v),w(w),  
        hlink(NULL),tlink(NULL){}  
};  
  
struct vexNode {  
    string data;  
    arcNode * first_in, * first_out;  
    vexNode(string data):data(data),first_in(NULL),first_out(NULL){}  
};  
  
struct OLGraph {  
    vector<vexNode> vexList;  
    void insertEdge(int u,int v,int w) {  
        arcNode * edge = new arcNode(u,v,w);  
        edge->tlink = vexList[u].first_out;  
        vexList[u].first_out = edge;  
        edge->hlink = vexList[v].first_in;  
        vexList[v].first_in = edge;  
    }  
    void printOutEdge(int u) {  
        arcNode * p = vexList[u].first_out;  
        while (p != NULL) {  
            cout << vexList[p->headvex].data << " -> ";  
            p = p->tlink;  
        }  
    }  
    void printInEdge(int u) {  
        arcNode * p = vexList[u].first_in;  
        while (p != NULL) {  
            cout << vexList[p->tailvex].data << " -> ";  
            p = p->hlink;  
        }  
    }  
};
```

# 3. 十字链表对于稀疏矩阵的构建
## 3.1 十字链表的结点
考虑一个 [[103-5 数组#3.5 十字链表下表示的稀疏矩阵 | 稀疏矩阵]]，此时我们用三元表来表示，当我们要插入一个元素的时候，此时用顺序表存储的三元表组就要向后移动，这个复杂度显然我们是希望优化掉的。
不妨考虑用十字链表来表示稀疏矩阵，此时假如需要插入一个新的元素，那么只需要修改四个指针。

对于十字链表上的一个结点，其包括五个部分：所在的行 `row` ，列 `col` ，数值 `val` ，右指针 `right` 和下指针 `down` 。
其中右指针指向在它右边的第一个非零元素，下指针指向在它下方的第一个非零元素。

此外，我们需要能进入这个十字链表的入口，因此十字链表需要维护两个指针数组：行头 `rhead[]` 和 `chead[]` 。
```cpp
struct OLNode {  
    int row, col;      
    int val;         
    OLNode *right, *down;      
    OLNode(int r, int c, int v) : row(r), col(c), val(v), right(NULL), down(NULL) {}  
};
```
## 3.2 十字链表的构建
对于一个矩阵，我们要维护其行数、列数和非零元素个数。
此外还有上文提及的两个指针数组。
#### 初始化
```cpp
int rows, cols, count; 
vector<OLNode*> rhead; // 行头
vector<OLNode*> chead; // 列头
  
// 初始化 
void init(int r, int c) {  
    rows = r; cols = c; count = 0;  
    rhead.assign(rows + 1, NULL);  
    chead.assign(cols + 1, NULL);  
}
```
#### 插入非零元
```cpp
void insert(int r, int c, int v) {  
    if (v == 0) return; // 0 值不写入  
  
    OLNode* p = new OLNode(r, c, v);  
    count++;  
  
    //将 p 插入到第 r 行的行链表中（保持列号递增）
    if (rhead[r] == NULL || rhead[r]->col > c) {  
        // 情况 1：行链表为空，p 应该成为该行的第一个节点  
        p->right = rhead[r];  
        rhead[r] = p;  
    }
    else {  
        // 情况 2：在行链表中找到 p 的前驱节点 pre
        OLNode* pre = rhead[r];  
        while (pre->right != NULL && pre->right->col < c) {  
            pre = pre->right;  
        }  
        p->right = pre->right;  
        pre->right = p;  
    }  
  
    //将 p 插入到第 c 列的列链表中（保持行号递增）   
    if (chead[c] == NULL || chead[c]->row > r) {  
        // 情况 1：列链表为空，p 应该成为该列的第一个节点  
        p->down = chead[c];  
        chead[c] = p;  
    } else {  
        // 情况 2：在列链表中找到 p 的前驱节点 pre
        OLNode* pre = chead[c];  
        while (pre->down != NULL && pre->down->row < r) {  
            pre = pre->down;  
        }  
        p->down = pre->down;  
        pre->down = p;  
    }  
}
```
## 3.3 完整结构
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

struct OLNode {
    int row, col;       // 行号、列号
    int val;            // 数值
    OLNode *right;      // 指向同行右侧下一个非零元
    OLNode *down;       // 指向同列下方下一个非零元

    OLNode(int r, int c, int v) : row(r), col(c), val(v), right(NULL), down(NULL) {}
};

struct CrossListMatrix {
    int rows, cols, count; // 矩阵行数、列数、非零元个数
    vector<OLNode*> rhead; // 行头指针向量
    vector<OLNode*> chead; // 列头指针向量

    // 初始化一个 rows * cols 的空矩阵
    void init(int r, int c) {
        rows = r; cols = c; count = 0;
        rhead.assign(rows + 1, NULL);
        chead.assign(cols + 1, NULL);
    }

    //插入一个非零元 (r, c, v)
    void insert(int r, int c, int v) {
        if (v == 0) return; // 0 值不写入

        OLNode* p = new OLNode(r, c, v);
        count++;

        // --- 步骤 A：将 p 插入到第 r 行的行链表中（保持列号递增） ---
        if (rhead[r] == NULL || rhead[r]->col > c) {
            // 情况 1：行链表为空，或者 p 应该成为该行的第一个节点
            p->right = rhead[r];
            rhead[r] = p;
        } else {
            // 情况 2：在行链表中找到 p 的前驱节点 pre
            OLNode* pre = rhead[r];
            while (pre->right != NULL && pre->right->col < c) {
                pre = pre->right;
            }
            p->right = pre->right;
            pre->right = p;
        }

        // --- 步骤 B：将 p 插入到第 c 列的列链表中（保持行号递增） ---
        if (chead[c] == NULL || chead[c]->row > r) {
            // 情况 1：列链表为空，或者 p 应该成为该列的第一个节点
            p->down = chead[c];
            chead[c] = p;
        } else {
            // 情况 2：在列链表中找到 p 的前驱节点 pre
            OLNode* pre = chead[c];
            while (pre->down != NULL && pre->down->row < r) {
                pre = pre->down;
            }
            p->down = pre->down;
            pre->down = p;
        }
    }
};
```
