# 1. 循环链表的构建和操作
循环链表也分为循环单链表和循环双链表，循环双链表是在循环单链表的基础上添加了 `prior` 指针。
因为链表是循环的，所以从链表的任意节点处出发均可以到达链表上的每一个结点。

在结点的定义上和 [[103-1 单链表]] 上没有区别，循环链表是通过将尾部结点的 next 指向了头结点实现的。

## 1.1 循环链表的初始化
空链表中，头节点 head 的 next 指针指向自己。
特殊地，在循环双链表中，还有 `head->prior = head` 。

```cpp
head = new Node;
head->next = head;
//head->prior = head;
```

同样地，我们可以通过判断 head 的 next（prior）是否指向自身来判断链表是否为空。

## 1.2 循环链表的操作变化
原本的链表中走到链表结尾判断 `p->next` 是否为 `nullptr` ，在循环链表中这个判断变为了判断 `p->next` 是否为 `head` 。

除此之外在指定位置操作也不需要特判尾结点了，因为在循环链表中默认不会有结点的指针域为空指针。

# 2. 循环链表的优化
可以在循环链表设置一个尾指针 rear，永远指向头节点的前驱，这样就能很方便地直接到达头节点。
