队列是一种先进先出（FIFO）的数据结构。

通常会使用封装好的 queue STL 来实现队列。

# 1. 队列的数组实现
接下来在数组上模拟队列，使用双指针来维护这个过程，一个指针指向队头 front，一个指针指向队尾 rear。

## 1.1 初始化和状态判断
初始时默认队列为空，front 和 rear 位置均为下标 0 。

```cpp
int que[MaxSize]
int front = 0,rear = 0;
```

当栈空的时候，有 `front == rear` 。

因为 front 和 rear 均是向右移动的，因此当有元素出栈的时候，会导致数组前面的空间被废弃，此时如果有 `rear == MaxSize` 则会变成 **假溢出** 的状态，并不代表数组的内存被完全使用完了。

要避免这个问题，可以给队列的每个元素动态分配内存，或者使用循环队列。

## 1.2 队列上的操作
#### 入队
```cpp
if(rear == MaxSize) return -1; //判满
que[++rear] = val;
```
#### 出队
```cpp
if(rear == front) return -1;
return que[front++];
```

## 1.3 出队序列计数
无论入队和出队操作顺序如何变化，队列的出队序列个数有且仅有 1 个，即原序列。

# 2. 循环队列

循环队列通过让 rear 和 front 在超过 MaxSize 的时候到达数组的另一端从而充分利用数组空间。
实现这一操作的过程是让下标对 MazSize 取模。

入队时 `rear = (rear+1)%MaxSize` ；
出队时 `front = (front+1)%MaxSize` 。

此时队列满的条件是 `(rear+1)%Maxsize == front` ，此时因为数组空间已经被完全利用了，因此此时是 **真溢出** 的状态。

循环链表中的元素个数为：

$$
(\mathrm{rear}-\mathrm{front}+siz)\%siz
$$
