栈是一种先进后出（LIFO）的数据结构，数据的存入和获取只能通过栈顶 top 操作。

通常会使用封装好的 stack STL 来实现栈。

# 1. 栈的数组实现
接下来在数组上模拟栈，使用一个指针来维护这个过程，该指针在栈顶 top。

## 1.1 初始化和状态判断
初始时默认栈空，此时 top 的值为 0 （从下标 1 开始存储数据）。

```cpp
int sta[MaxSize];
int top = 0;
```

根据我们初始化的状态，当栈空时，`top == 0` ，当栈满时，top 的值为数组的大小，一般会比实际大小小一些。
## 1.2 入栈和出栈
#### 入栈
```cpp
if(top == Maxsize) return false; //栈满则不能入栈
sta[++top] = val;
```
#### 出栈
```cpp
if(top == 0) return -1; //栈空则不能出栈
return sta[top--];
```

这些操作的时间复杂度均为 $O(1)$ 。

# 2. 出栈序列计数
给定 $n$ 个不同的元素，让他们依次入栈，合法的出栈序列个数为：

$$
C_{n}=\frac{1}{n+1}\binom{2n}{n}
$$

这是一个卡特兰数。

# 3. 共享栈
共享栈指的是两个栈共用一个数组的空间，栈 1 的指针 top1 从左向右移动，栈 2 的指针 top2 从右向左移动，当两个指针相遇（`top1+1 == top2`）的时候栈满。

# 4. 栈在递归中的运用

递归的工作原理基于系统内的递归工作栈，每次调用函数时，系统会分配一个内存（栈帧）存储返回地址、局部变量和参数值。

当递归深度过大的时候，栈空间可能会耗尽。
