# 1. 离散化

对于一般的数据，我们当然可以排序求出他们的关系，但是我们也可以不关注他们的值，只关注他们的相对大小关系，我们就可以将他们映射到一个连续的更小的整数区间内，即离散化。

离散化最重要的作用就是把大数化小，只关注他们的相对关系值。

具体的操作如下：
1. 对数组排序
2. 除去重复元素
最后对应的数组下标就是各个数间的相对关系。
对于排序很简单，sort就行了。而除去重复元素我们就需要用一个新的函数：unique。

# 2. unique() 
unique函数会对两个相邻的相同数据进行去重，将重复的元素移至数组后方。

通过上面的描述我们需要知道unique函数的性质：

1. unique函数不会对数组的大小进行改变；
2. unique函数只比较相邻的元素；
3. unique函数返回的是一个迭代器（指针），地址位于去重后的数组的逻辑结尾。

对于第2点，我们在使用unique函数的之前需要进行排序处理：
```cpp
{1,2,1,3} --unique-> {1,2,1,3} //可以发现并没有进行去重

{1,1,2,3} --unique-> {1,2,3,1} //可以发现进行了去重，将重复的1放到了数组末尾，返回的迭代器指向数组中元素为3的位置
```

对于第3点，我们在使用的时候如果想知道不重复元素的个数，就需要减去数组的首地址，考虑到下标问题，需要减去 (数组首地址+1) 的值。

# 3. 实现

#### 顺序表内实现
```cpp
#include <bits/stdc++.h>  
using namespace std;  
typedef long long ll;  
  
int t;  
int a[100005],d[100005];  
  
int main() {  
    cin >> t;  
    while (t--) {  
        memset(a, 0, sizeof a);  
        int n; cin >> n;  
        for (int i = 1 ; i <= n ; i++) cin >> a[i] , d[i] = a[i];  
        sort(a+1,a+1+n);  
        int last = unique(a+1,a+1+n) - (a+1);  
        for (int i = 1 ; i <= n ; i++) {  
            cout << lower_bound(a+1,a+last+1,d[i])-a << " ";  
        }  
        cout << endl;  
    }  
}
```

#### vector 方法实现
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int t;
int a[100005];
vector<int> lib;

int main() {
    cin >> t;
    while (t--) {
        lib.clear();
        lib.push_back(-0x3f3f3f3f3f3f3f);
        int n; cin >> n;
        for (int i = 1 ; i <= n ; i++) {
            cin >> a[i];
            lib.push_back(a[i]);
        }
        sort(lib.begin(), lib.end());
        auto pend = unique(lib.begin(), lib.end());
        lib.erase(pend,lib.end());
        for (int i = 1 ; i <= n ; i++) {
            cout << lower_bound(lib.begin(),lib.end(),a[i]) - lib.begin() << ' ';
        }
        cout << endl;
    }
}
```
