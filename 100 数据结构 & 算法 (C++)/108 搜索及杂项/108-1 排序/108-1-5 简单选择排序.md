# 1. 简单选择排序 $\mathbf{SELECT\;SORT}$
对于排序表 $L[1,2,\dots,n]$ ，第 $i$ 趟排序是从 $L[i,\dots,n]$ 中选择关键字最大/小的元素与 $L[i]$ 交换。
排序趟数为 $n-1$ 。

#### 代码实现
```cpp
#include <bits/stdc++.h>  
using namespace std;  
#define int long long  
#define detristy ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);  
  
const int N = 100010;  
int a[N], n;  
  
void select_sort() {  
    for (int i = 1 ; i <= n ; i++) {  
        int mn_idx = i;  
        for (int j = i+1 ; j <= n ; j++) {  
            if (a[j] < a[mn_idx]) {  
                mn_idx = j;  
            }  
        }  
        if (mn_idx != i) swap(a[i],a[mn_idx]);  
    }  
}  
  
void solve() {  
    cin >> n;  
    for (int i = 1; i <= n; i++) cin >> a[i];  
    select_sort();  
    for (int i = 1; i <= n; i++) cout << a[i] << ' ';  
    cout << "\n";  
}  
  
signed main() {  
    detristy  
    int T = 1;  
    //cin >> T;  
    while (T--) solve();  
    return 0;  
}
```

# 2. 复杂度分析
最坏情况和最好情况和平均情况的时间复杂度均为 $O(n^2)$ 。
空间复杂度为 $O(1)$ 。

# 3. 稳定性
因为涉及交换操作，可能会导致相同元素相对位置发生改变，故而是不稳定的。
