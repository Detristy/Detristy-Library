# 1. 归并排序 $\mathbf{MERGE\;SORT}$

归并排序同样是基于**分治**思想的排序算法：

1. **分**：将待排序序列从中间分成左右两个子序列。
2. **治**：递归地对左右两个子序列分别进行归并排序。
3. **合**：将两个已经有序的子序列**合并**（merge）成一个有序序列。

归并排序的核心在于"合"这一步——如何高效地将两个有序数组合并为一个有序数组。

#### 合并过程
使用双指针 $i$ 和 $j$ 分别指向左右两个有序子序列的起始位置，每次比较 `a[i]` 和 `a[j]`，将较小者放入临时数组，对应指针后移。当其中一个子序列遍历完后，将另一个子序列剩余元素直接追加到临时数组末尾。

#### 代码实现

```cpp
const int N = 100010;
int tmp[N];  // 辅助数组，用于合并

void merge_sort(int a[], int l, int r) {
    if (l >= r) return;
    int mid = (l + r) >> 1;
    
    // 递归排序左右两部分
    merge_sort(a, l, mid);
    merge_sort(a, mid + 1, r);
    
    // 合并两个有序序列
    int i = l, j = mid + 1, k = 0;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else              tmp[k++] = a[j++];
    }
    while (i <= mid) tmp[k++] = a[i++];  // 左半部分剩余
    while (j <= r)   tmp[k++] = a[j++];  // 右半部分剩余
    
    // 将合并结果写回原数组
    for (i = l, k = 0; i <= r; i++, k++) a[i] = tmp[k];
}
```

# 2. 性能分析

#### 时间复杂度
归并排序的递归树深度始终为 $\lceil \log_2 n \rceil$，每层合并操作的复杂度为 $O(n)$。

- **最好情况**：$O(n\log n)$
- **最坏情况**：$O(n\log n)$
- **平均情况**：$O(n\log n)$

归并排序的时间复杂度与输入数据的初始顺序**无关**，性能非常稳定。

#### 空间复杂度
合并时需要借助长度为 $n$ 的辅助数组 `tmp`，再加上递归栈的深度 $O(\log n)$，总空间复杂度为 $O(n)$。

> 归并排序不是原地排序，这是它相对于快速排序的一个劣势。但它适合处理**外部排序**（数据量大到无法全部装入内存时，可以分段归并）。

# 3. 稳定性

在合并时，当 `a[i] == a[j]` 时我们优先取左边子序列的元素（`a[i] <= a[j]` 的条件下取 `a[i]`），因此相同元素的相对顺序得以保留。归并排序是**稳定**的。

# 4. 应用——逆序对计数

归并排序的合并过程可以顺便统计逆序对数量。当 `a[i] > a[j]` 时，由于左半部分 `[i, mid]` 的元素都大于 `a[j]`，因此产生了 `mid - i + 1` 个逆序对。

```cpp
long long merge_sort_count(int a[], int l, int r) {
    if (l >= r) return 0;
    int mid = (l + r) >> 1;
    long long res = merge_sort_count(a, l, mid) 
                  + merge_sort_count(a, mid + 1, r);
    
    int i = l, j = mid + 1, k = 0;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else {
            tmp[k++] = a[j++];
            res += mid - i + 1;  // 统计逆序对
        }
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r)   tmp[k++] = a[j++];
    for (i = l, k = 0; i <= r; i++, k++) a[i] = tmp[k];
    return res;
}
```
