# 1. 树的带权路径长度

在介绍霍夫曼树之前，我们引入 [[102-1 树]] 的带权路径长度这个概念。
>[!definition] 定义 1
> 设 $w_{i}$ 为二叉树第 $i$ 个叶子节点的权值， $l_{i}$ 为从根节点到达第 $i$ 个叶子结点的路径长度，则定义树的带权路径长度 $WPL$ 为：
>$$
>	WPL=\sum\limits_{i=1}^nw_{i}l_{i}
>$$

# 2. 霍夫曼树及其构造
## 2.1 霍夫曼树
>[!definition] 定义 2
> 霍夫曼树是 WPL 最小的二叉树。
> ^def-7cb9b9

霍夫曼树是一颗二叉树，因此它也会有一些独特的性质：

> [!tip] 性质 1
> 具有 $n$ 个权值的霍夫曼树的结点个数为 $2n-1$ 。

## 2.2 霍夫曼算法

考虑贪心，对于一颗霍夫曼树而言，我们要尽量在二叉树的基础上，保证权值越小的叶子离根更远，权值越大的叶子离根越近。

霍夫曼树的构造通过霍夫曼算法，大致步骤如下：
1. 由给定的 $n$ 个权值构造 $n$ 棵只有一个根结点的二叉树，得到一个二叉树集合。
2. 从二叉树集合中选取根结点权值最小的两棵二叉树分别作为左右子树构造一个新的二叉树，新二叉树的根节点权值为左右子树的根节点的权值之和。
3. 删除刚刚的两棵二叉树，将新的二叉树放入集合中。
4. 重复2，3直到所有的结点都被加入树中。

我们定义树上的结点如下：

```cpp
struct node{
	int w,id; //w 是权值，id 是结点编号
};
```

我们可以用一个通过小根堆维护的优先队列 $q$ 来存储每一个结点，故而上述过程可以被下面的代码模拟：

```cpp
while (q.size() > 1) {
	node x = q.top(); 
	q.pop(); 
	node y = q.top(); 
	q.pop(); 
	cnt++; 
	e[cnt] = {x.id,y.id}; 
	q.push({x.w+y.w,cnt}); 
}
```

优先队列保证了每一次取出的结点 $x$ 和 $y$ 是最小的。

# 霍夫曼编码

霍夫曼编码是一种可变长前缀编码，它通过构建哈夫曼树来实现，使得 $WPL$ 最小。运用霍夫曼树实现霍夫曼编码的时候，霍夫曼树的左分支通常代表 0，右分支代表 1 。

我们统计所有单词中每个字符或单词出现的频次，将其分配至霍夫曼树上，接着为其分配编码。

特殊地，在单词或字符个数为 1 的时候，只需要分配一个编码 0 即可。

大致实现如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long

constexpr int N = 4005;
int n,cnt;
pair<int,int> e[N];
string s[N],chain[N],cur="";

struct node {
    int w,id;
    bool operator<(node a) const {
        return w > a.w;
    }
};

priority_queue<node> q;

void dfs(int u) {
    if (u <= n) {
        chain[u] = cur;
        return;
    }
    cur += '0';
    dfs(e[u].first);
    cur.pop_back();
    cur += '1';
    dfs(e[u].second);
    cur.pop_back();
}

void solve() {
    cin >> n;
    cnt = n;
    for (int i = 1 ; i <= n ; i++) {
        int w;
        cin >> s[i] >> w;
        q.push({w,i});
    }
    if ( n == 1 ) {
        cout << s[1] << ' ' << "0";
        return;
    }
    while (q.size() > 1) {
        node x = q.top();
        q.pop();
        node y = q.top();
        q.pop();
        cnt++;
        e[cnt] = {x.id,y.id};
        q.push({x.w+y.w,cnt});
    }
    dfs(cnt);
    for (int i = 1 ; i <= n ; i++) {
        cout << s[i] << ' ' << chain[i] << '\n';
    }
}

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    int T = 1;
    //cin >> T;
    while (T--) solve();
    return 0;
}
```
