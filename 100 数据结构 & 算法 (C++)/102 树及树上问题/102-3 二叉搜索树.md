二叉搜索树是一种二叉 [[102-1 树]] 的数据结构。

# 1. 定义
>[!definition] 定义
> 1. 空树是二叉搜索树。
> 2. 若二叉搜索树的左子树不为空，则其左子树上所有点的权值均小于其根节点。
> 3. 若二叉搜索树的右子树不为空，则其右子树上所有点的权值均大于其根节点。
> 4. 二叉搜索树的子树是二叉搜索树

二叉搜索树的期望高度为 $O(\log N)$。

# 2. 构造

二叉搜索树本质上是一棵二叉树，我们使用左右儿子表示法来构造。

```cpp
struct TreeNode {  
    int val;  
    TreeNode *left;  
    TreeNode *right;  
    TreeNode(int value) : val(value), left(NULL), right(NULL) {}  
};
```

这里用一个构造函数为节点内的变量进行初始化，左右儿子默认初始化为空指针。

# 3. 二叉搜索树上操作

## 3.1 遍历

二叉搜索树常用中序遍历，由二叉搜索树的定义可得中序遍历的结果是一个非降的序列。

```cpp
void inorder(TreeNode *root) {  
    if (root == NULL) return;  
    inorder(root->left);  
    cout << root->val << ' ';  
    inorder(root->right);  
}
```

复杂度是 $O(N)$ 。

## 3.2 搜索

插入，删除和修改操作都需要先进行搜索操作。

我们的目标是查找一个值为 value 的结点。

根据定义，我们分类讨论，设 $r$ 为当前子树的根节点：
显然地，如果值为 value 返回 true，如果 $r$ 为空返回 false；
如果 $r$ 的权值大于 value ，搜索左子树；
如果 $r$ 的权值小于 value ，搜索右子树。

```cpp
bool find(TreeNode *root, int value) {  
    if (root == NULL) return false;  
    if (root->val == value) return true;  
    if (root->val > value) return find(root->left, value);  
    return find(root->right, value);  
}
```

## 3.3 找最值

由定义，二叉搜索树的最小值为左子树最左端，最大值为右子树最右端，找起来比较简单，这里以查找最小值为例。

```cpp
TreeNode* findMinNode(TreeNode* root) {  
    while (root->left != nullptr) {  
        root = root->left;  
    }  
    return root;  
}
```

## 3.4 插入

我们的目标是在以 $r$ 为根节点的二叉搜索树中插入一个值为 value 的结点。

进行分类讨论，设 $r$ 为当前子树的根节点：
若 $r$ 为空，直接返回一个值为 value 的新节点；
若 $r$ 的权值大于 value，这个新结点的位置则位于 $r$ 的左子树中，在左子树中插入；
若 $r$ 的权值小于 value，则在右子树中插入。

```cpp
TreeNode* insert(TreeNode *root,int value) {  
    if (root == NULL) {  
        return new TreeNode(value);  
    }  
    if (root->val > value) root->left = insert(root->left, value);  
    else root->right = insert(root->right, value);  
    return root;
}
```

## 3.5 删除

删除比较麻烦。

我们的目标是在以 $r$ 为根节点的二叉搜索树中删除一个值为 value 的结点。

我们先对 value 进行搜索，找到权值为 value 的结点，然后分类讨论执行删除操作：

如果这个结点是叶子结点，我们可以直接删除而不造成其他影响。
如果这个结点只有一个儿子，那么直接把儿子连接到这个结点的父节点上。
如果这个结点有两个儿子，一般是用其左子树的最大值或右子树的最小值代替当前结点，这里我们可以考虑直接赋值，然后对其删除，这里以右子树的最小值代替为例。

```cpp
TreeNode* remove(TreeNode *root,int value) {  
    if (root == NULL) return root;  
    if (value < root->val) root->left = remove(root->left, value);  
    else if (value > root->val) root->right = remove(root->right, value);  
    else {  
        if (root->left == NULL) {  
            TreeNode *tmp = root->right;  
            delete root;  
            return tmp;  
        }  
        else if (root->right == NULL) {  
            TreeNode *tmp = root->left;  
            delete root;  
            return tmp;  
        }  
        else {  
            TreeNode *mnNode = findMinNode(root->right);  
            root->val = mnNode->val;  
            root->right = remove(root->right,mnNode->val);  
        }  
    }  
    return root;  
}
```

# 4. 局限性

## 4.1 不唯一性

二叉搜索树的建立与输入顺序有关，也就是说同一组数据的不同排列会构造出不同的二叉搜索树，即一组数据的二叉搜索树不是唯一的。

## 4.2 退化

二叉搜索树的思想和二分搜索很相近，但是当输入的数据趋于有序的时候，二叉搜索树会逐渐变成一条链，导致其逐渐变得和按顺序遍历很相似，损失了时间复杂度优势。

这被称之为二叉搜索树的退化，这个问题可以通过引入平衡因子，构造平衡树来解决。

# 5. 实现例
>[!example] 例题
>给定一个数列，构建相应的二叉排序树，并利用其中序遍历对数据进行升序排序。
>
>**输入：**
>一行，第一个数 $N$ 代表数据个数，第 $i+1$ 到第 $i+N$ 个数据代表 $a_{i}$ 。
>
>**输出：**
>一行，中序遍历后的数据。

```cpp
#include <bits/stdc++.h>  
using namespace std;  
#define int long long  
#define detristy ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);  
  
struct TreeNode {  
    int val;  
    TreeNode *left;  
    TreeNode *right;  
    TreeNode(int value) : val(value), left(NULL), right(NULL) {}  
};  
  
void inorder(TreeNode *root) {  
    if (root == NULL) return;  
    inorder(root->left);  
    cout << root->val << ' ';  
    inorder(root->right);  
}  
  
TreeNode* insert(TreeNode *root,int value) {  
    if (root == NULL) {  
        return new TreeNode(value);  
    }  
    if (root->val > value) root->left = insert(root->left, value);  
    else root->right = insert(root->right, value);  
    return root;  
}  
  
void solve() {  
    int n; cin >> n;  
    TreeNode * root = NULL;  
    for (int i = 1 ; i <= n ; i++) {  
        int val;  
        cin >> val;  
        root = insert(root,val);  
    }  
    inorder(root);  
}  
  
signed main() {  
    detristy

    //cin >> T;

    return 0;  
}
```
