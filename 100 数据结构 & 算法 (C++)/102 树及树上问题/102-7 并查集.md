并查集能够维护各个结点是在连通块内的状态，包括查询和合并操作。

# 1. 并查集
## 1.1 并查操作
并查集由两部分组成：一个是合并 Union ，一个是查找 Find。

很明显，并就是把两个独立的集合合并成一个。

查就是查询两个元素是否属于同一个集合，在树里就相当于问是否属于同一个根节点，我们默认接下来的“查”操作都是在树里进行查找。

在我们开始操作之前，我们需要对每一个元素进行初始化，即自己是属于自己的父亲。

```cpp
fa[i]=i //fa代表father
```

接下来查，查就是查询 `fa[i]` 的值，如果 `fa[x] == x`，那么自己就是根节点，当然属于这个集合

如果 `fa[x] == y` ，那么我们就要传递一次，去查询 `fa[y]` ，直到找到根节点

然后是并，并就是将两个集合合并，这里假设将X和Y合并，那么我们需要找到X和Y的根节点 root_x 和 root_y，如果不相等，那就让 root_y归到 root_x 的子节点处（反过来也一样），这样就有 `fa[root_y] == root_x`

但是我们暴力查找的复杂度是 $O(N)$ 起步，很慢，因此我们需要进行路径压缩。
## 1.2 路径压缩
我们假设确定了点 $x$ 的根节点是 $root$，那么下一次查点 $x$ 的时候就不需要继续搜索了，我们直接干脆写 `fa[x]=root` 即可实现 $O(1)$ 搜索

很明显，我们前提是得知道 $x$ 的根节点是 $root$，那么还是避免不了搜一次，但是我们搜一次肯定会遍历一堆中间节点（假设是 $a_i$ ）那我们不如直接在搜索的时候直接 `fa[a_i]=root` ，于是这样路径上所有的点的父节点就变成了$root$，整棵树被压到只剩两层，这样的话除了第一次查询，后面的查询几乎都是 $O(1)$ 的。

## 1.3 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int n,m;
const int N = 1e6+10;
int fa[N];

//查的过程，用递归的方式实现压缩
int find(int x) {
    if (fa[x]==x) {
        return x;
    }
    fa[x] = find(fa[x]);
    return fa[x];
    //其实也可以完全用三目运算符写成：
    //return fa[x] == x ? x : fa[x] = find(fa[x])
}

//合并的过程
void uni(int x,int y) {
    int fx = find(x);
    int fy = find(y);
    if (fx!=fy) {
        fa[fx] = fy;
    }
}

int main() {
    cin >> n >> m;
    
    //初始化
    for (int i = 1 ; i <= n ; i++) {
        fa[i] = i;
    }
    
    for (int i = 1 ; i <= m ; i++) {
        int z,x,y;
        cin >> z >> x >> y;
        if (z==1) {
            uni(x,y);
        }
        if (z==2) {
            if (find(x) == find(y)) {
                cout << "Y" << endl;
            }
            else cout << "N" << endl;
        }
    }
    return 0;
}
```

# 2. 带权并查集
带权并查集上每一个点 $i$ 都带有一个权重 $d$ ，这个 $d$ 代表点 $i$ 此时与其根节点 `fa[i]`  的相对关系。我们需要在路径压缩的时候调整这个关系即：$$
	x→root=(x→father)+(father→root)
$$可以看作向量的加法。
在合并的时候，假如我们需要合并两个集合 X 和 Y：

x 是 X 的一个节点，根节点是root_x；
y 是 Y 的一个节点，根节点是root_y；

我们合并两个集合，很明显就是要让 $fa[root_x] = root_y$ （反过来也成立）。

问题在于，我们怎么确定 $root_x$ 和 $root_y$ 之间的距离呢？

我们就可以利用刚刚的那个关系：
$$
	 (x\to root_x)+(root_x\to root_y)=(x\to y)+(y\to root_y)
$$
此时我们知道 $d[x]$ ，$d[y]$ 以及 $x$ 到 $y$ 的距离 $w$，那么 $root_x$ 到 $root_y$ 的距离我们就可以通过移项得到：
$$
d[root_x] = w + d[y]-d[x]
$$
