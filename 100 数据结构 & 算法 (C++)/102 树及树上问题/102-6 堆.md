# 1. 堆
堆是一棵树，树上每一个结点都有对应的键值，每个结点的键值都大于等于或小于其父节点的键值。
# 2. 二叉堆
## 2.1 二叉堆的性质
二叉堆是一棵完全二叉树，分为两类：大根堆和小根堆。
大根堆中父亲的权值不小于儿子的权值，小根堆中父亲的权值不大于儿子的权值。
下文我们主要以小根堆为例。
## 2.2 二叉堆的映射
因为二叉堆是一棵完全二叉树，因此我们可以很方便地将其映射到数组上。
对于索引从 1 开始的数组，结点 $i$ 的父亲为 $\left\lfloor  \frac{i}{2}  \right\rfloor$ ，其左孩子结点为 $2i$ ，右孩子结点为 $2i+1$ 。
# 3. 堆上操作 
## 3.1 堆上调整
堆上的调整由两个动作构成，分别是下沉和上浮。
#### 下沉
对于小根堆上某个结点的值，如果突然变大，则理论上而言其需要向下移动，我们将这个移动的操作称为下沉 (Down) 。
对于下沉操作而言我们要考察当前结点的左儿子和右儿子对应的权值是否比当前结点小，如果是则考虑将当前结点和左/右儿子进行交换，直到到达叶子结点或没有比它更小的结点为止，这是一个递归的过程。
```cpp
void down(int u) {  
    int t = u;  
    if (u*2 <= sz && hp[u*2] < hp[t]) t = u*2;  
    if (u*2+1 <= sz && hp[u*2+1] < hp[t]) t = u*2+1;  
    if (u != t) {  
        swap(hp[u],hp[t]);  
        down(t);  
    }  
}
```
#### 上浮
对于小根堆上某个结点的值，如果小于其父亲结点，则理论上而言需要向上移动，我们将这个移动操作称为上浮 (Up) 。
对于上浮操作而言，只需要考虑其父亲即可。
```cpp
void up(int u) {  
    while (u/2 > 0 && hp[u] < hp[u/2]) {  
        swap(hp[u],hp[u/2]);  
        u /= 2;  
    }  
}
```
## 3.2 堆上插入
对于堆上插入一个新的结点，要保证其插入后也是一棵完全二叉树。
故考虑在最下一层最右边的叶子后面插入，如果最下一层已经满了，就向下开一层。

最好的情况下，插入的数据能够刚好符合小/大根堆的性质。
而在其他情况下，插入的结点可能会不满足堆的性质，因此我们考虑对堆上结点进行调整，即强行将其上浮，直到找到对应合适的位置。
```cpp
void push(int x) {  
    hp[++sz] = x;  
    up(x);  
}
```
## 3.3 堆上删除
此处的堆上删除指的是弹出堆上最小的结点，此时堆的大小 -1。

显然删除的结点对应小根堆上的根节点，我们是需要对堆进行调整的，具体的操作是将堆的最后一个结点移动到根结点，接着当前的堆进行调整，即强行将其下沉，直到找到合适的位置。
```cpp
int pop() {  
    int u = hp[1];  
    hp[1] = hp[sz--];  
    down(1);  
    return u;  
}
```
## 3.4 建堆
我们只需要考虑从 $\lfloor \frac{2}{n} \rfloor$ 处开始对每一个元素进行下沉，就可以将任意数组转化为堆。时间复杂度为 $O(n)$ 。
```cpp
void buildHeap() {  
    for (int i = sz/2 ; i ; i--) down(i);  
}
```
## 3.5 堆结构
```cpp
struct heap {  
    int hp[N];  
    int sz;  
  
    void down(int u) {  
        int t = u;  
        if (u*2 <= sz && hp[u*2] < hp[t]) t = u*2;  
        if (u*2+1 <= sz && hp[u*2+1] < hp[t]) t = u*2+1;  
        if (u != t) {  
            swap(hp[u],hp[t]);  
            down(t);  
        }  
    }  
    void up(int u) {  
        while (u/2 > 0 && hp[u] < hp[u/2]) {  
            swap(hp[u],hp[u/2]);  
            u /= 2;  
        }  
    }  
    void push(int x) {  
        hp[++sz] = x;  
        up(x);  
    }  
    int pop() {  
        int u = hp[1];  
        hp[1] = hp[sz--];  
        down(1);  
        return u;  
    }  
    void buildHeap() {  
        for (int i = sz/2 ; i ; i--) down(i);  
    }  
}h;
```

# 4. 堆的局限性
堆的性质是很弱的，因此对于同一组数，不同的输入顺序会导致建出来的堆不同，也就是说二叉堆并不是唯一的。
堆只能保证堆顶（根节点）是极值，不能保证除了根节点外的其他结点是有序的。
