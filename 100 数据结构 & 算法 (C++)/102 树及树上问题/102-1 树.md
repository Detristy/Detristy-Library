# 1. 树
## 1.1 树的概念
树是一种递归的数据结构，能够表示数据的层级结构。接下来引入与树有关的术语。
#### 层次
结点的层次从根节点处开始定义，根节点为第 1 层，结点的深度就是其所在的层次。
#### 树的高度
树的高度是树上所有结点的层次最大值。
#### 度
结点的度是结点的儿子个数。树的度是所有结点度的最大值。
#### 叶子
度为 0 的结点。
#### 分支
度大于 0 的结点。
#### 森林
森林是 $m$ 棵互不相交的树的集合。
## 1.2 树的性质
树的结点数、高度、度之间互有关系：
>[!corollary] 性质 1
>树的结点数 = 所有点的度数和 + 1

>[!corollary] 性质 2
>度为 $m$ 的树上第 $i$ 层的结点数至多为 $m^{i-1}$ $(i\geq 1)$。

>[!corollary] 性质 3
>高度为 $h$ 的 $m$ 叉树上的结点个数至多为：
>

>$$
>	\frac{m^h-1}{m-1}
>$$

性质 3 的证明是显然的，要使得结点个数最多，保证每一层都被填满，那么结点个数为：

$$
1+m+m^2+\dots+m^{h-1}=\frac{m^h-1}{m-1}
$$

>[!corollary] 性质 4
>度为 $m$ ，结点数为 $n$ 的树的最小高度 $h_{min}$ 为：
>

>$$
>	h=\lceil \log_{m}[n(m-1)+1] \rceil 
>$$

考虑贪心：要使得高度最小，则对应地要把每一层都填满。
由性质 3，假设将前 $h-1$ 层均填满，则结点个数最多为

$$
\frac{m^{h-1}-1}{m-1}
$$

那么前 $h$ 层的结点个数 $n$ 满足：

$$
\frac{m^{h-1}-1}{m-1} < n \leq \frac{m^{h}-1}{m-1}
$$

把 $h$ 的值反解出来即可。

# 2. 二叉树
二叉树的每一个结点至多有两个分支，第 n 层具有的节点个数至多为 $2^{n-1}$ 个。
## 2.1 二叉树结点的定义
二叉树结点由数据域，和分别指向左节点和右节点的两个指针。
```cpp
struct node{
	int l,r;
	int data; //假设存储一个整形数据
}tree[N] //存储N个这样的节点
```

## 2.2 二叉树的遍历
### 2.2.1 先序遍历
先序遍历的遍历顺序是 左子树->根节点->右子树。
```cpp
void PreOrder(TNode * r) {  
    if (r == NULL) return;  
    cout << r->val;  
    PreOrder(r->l);  
    PreOrder(r->r);  
}
```
### 2.2.2 中序遍历
先序遍历的遍历顺序是 根节点->左子树->右子树。
```cpp
void InOrder(TNode * r) {  
    if (r == NULL) return;  
    PreOrder(r->l);  
    cout << r->val;  
    PreOrder(r->r);  
}
```
### 2.2.3 后序遍历
先序遍历的遍历顺序是 左子树->右子树->根节点。
```cpp
void InOrder(TNode * r) {  
    if (r == NULL) return;  
    PreOrder(r->l);  
    cout << r->val;  
    PreOrder(r->r);  
}
```
### 2.3.4 层序遍历
层序遍历是通过队列实现的，本质上是 BFS。
```cpp
void FloorOrder(TNode * r) {    
    queue<TNode> q;  
    q.push(r);  
    while (!q.empty()) {  
        TNode rt = q.top();  
        q.pop();  
        cout << rt->val << ' ';  
        q.push(rt->l);  
        q.push(rt->r);  
    }  
}
```
## 2.3 遍历序列构造二叉树
### 2.3.1 输入先序遍历构造二叉树
先从根节点出发，因为根节点的左右儿子分别是其左右子树的根节点，因此给定一个先序遍历我们可以递归构造出一个二叉树。

使用这个方法的前提是叶子结点的位置是已知的，下面默认权值为 0 的点是空结点。

```cpp
TNode * BuildTree() {  
    int value;  
    if ( !(cin >> value) || value == 0) {  
        return NULL;  
    }  
    TNode * r = new TNode(value);  
    r->l = BuildTree();  
    r->r = BuildTree();  
    return r;  
}
```

### 2.3.2 中序序列和先/后序序列确定二叉树
当给定中序序列后，只要知道先序序列和后序序列其中的任意一个就可以将二叉树确定下来。

以先序序列为例，我们能够知道先序序列的第一个结点必定是根节点，因此我们在中序序列找到对应的结点就可以将中序序列分为左右两半，分别是左子树和右子树，对于每一个子树也可以在先序序列里进行分割，以此递归下去即可得到一棵完整的二叉树。

需要注意的是如果只知道先序序列和后序序列是无法确定一个二叉树的。

# 3. 一般树
接下来我们讨论对于一般的非二叉树，其存储和树上操作应该如何实现。
## 3.1 树的存储
因为树的叉数我们无法确定，因此到这里我们就不能单纯地使用左右指针来存储树了。
### 3.1.1 双亲表示法
双亲表示法指的是对于每个结点，我们记录其父亲的位置。不妨考虑采用顺序存储，这样著需要记录父亲结点所在的下标即可（伪指针）。

```cpp
struct TNode {  
    int data;  
    int parent;  
};  
  
struct Tree {  
    TNode nodes[MAX_SIZE];  
    int n;  //树上的结点个数
};
```

### 3.1.2 邻接表（孩子表示法）
将树按照图的结构进行存储，对于一个结点，为其分配一个顺序表或者链表来存储其指向所有儿子的指针。方便参考，这里提供一个 vector 的实现方式。

```cpp
vector<int> g[MAX_SIZE];
//或者
vector<vector<int>> g;
```

### 3.1.3 孩子兄弟表示法
以一个二叉链表作为树的存储结构，使用孩子兄弟表示法时，除了数据域之外，要为结点分配两个指针-：指向结点的第一个孩子结点的指针，指向结点下一个兄弟的指针。

```cpp
struct CSNode{
	int data;
	CSNode * firstChild;
	CSNode * nextBro;
}
```

## 3.2 一般树的遍历
一般非空树的遍历分为先根遍历、后根遍历和层序遍历。
#### 先根遍历
先根遍历是先遍历当前子树的根，再依次访问根节点的每一棵子树，对于每一棵子树也采用先根遍历的规则递归遍历。
#### 后根遍历
后根遍历是先遍历当前子树的每一棵子树，最后访问当前子树的根。
#### 层序遍历
按层次访问当前层次的所有结点，可以使用队列实现，也可以用孩子兄弟表示法实现。

# 4. 树间关系
## 4.1 树转换为二叉树
规则：另每一个结点的左指针指向其第一个儿子，右指针指向它在数上的相邻的右兄弟。

需要注意，因为根节点没有兄弟结点，因此在转换结束之后，根节点不存在右子树。

## 4.2 森林转换为二叉树
规则：先将森林里的所有树都转换为二叉树，将所有二叉树的根节点相连。

因为对于所有的转换后的二叉树都没有右子树，因此能够保证最后得到的树上每一个结点的度不大于2。

## 4.3 二叉树转换为森林
由 [[#4.2 森林转换为二叉树]]  可知，根节点的每一个右子树的根节点就对应另一棵树的根节点，因此只需要将树的根的右链递归地断开即可。
