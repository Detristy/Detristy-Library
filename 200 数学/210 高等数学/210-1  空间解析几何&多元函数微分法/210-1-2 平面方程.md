# 1. 平面的方程

确定一个平面需要两个条件：**一个法向量** + **平面上的一点**。这与 [[210-1-1 向量及其运算|向量的数量积]] 紧密相关——平面方程本质上就是利用"平面上任意向量与法向量垂直"这一几何条件推导出来的。

## 1.1 点法式方程
>[!definition] 定义（点法式方程）
> 设平面 $\Pi$ 的法向量为 $\mathbf{n} = (A, B, C)$，且过点 $M_0(x_0, y_0, z_0)$，则对平面上任意一点 $M(x, y, z)$，有 $\overrightarrow{M_0M} \perp \mathbf{n}$，即：

> $$
> A(x - x_0) + B(y - y_0) + C(z - z_0) = 0
> $$

> 此即平面的**点法式方程**。

点法式方程的本质是 $\mathbf{n} \cdot \overrightarrow{M_0M} = 0$——利用 [[210-1-1 向量及其运算#^perpendicular-condition|向量垂直的充要条件]] 直接得到。

## 1.2 一般方程
将点法式展开，所有常数项合并，即得平面的一般方程：
>[!definition] 定义（一般方程）

> $$
> Ax + By + Cz + D = 0
> $$

> 其中 $A, B, C$ **不全为零**，且 $(A, B, C)$ 即为平面的一个法向量。

可以推出一般方程中系数的几何含义：

- $D = 0$：平面**过原点**；
- $A = 0$：法向量垂直于 $x$ 轴 $\implies$ 平面**平行于 $x$ 轴**；
- $A = B = 0$：平面同时平行于 $x$ 轴和 $y$ 轴 $\implies$ 平面**平行于 $xOy$ 面**（即垂直于 $z$ 轴）。

> [!tip] 快速判断平面特点
> 缺哪个变量，平面就平行于哪个坐标轴。例如：
> - $3y - 1 = 0$：只含 $y$，平行于 $xOz$ 面（垂直于 $y$ 轴）；
> - $x - 3y = 0$：缺 $z$，平行于 $z$ 轴且过原点。

## 1.3 截距式方程
若平面在 $x$、$y$、$z$ 轴上的截距分别为 $a$、$b$、$c$（均不为零），则：
>[!definition] 定义（截距式方程）

> $$
> \frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1
> $$

事实上，只需将三点 $(a,0,0)$、$(0,b,0)$、$(0,0,c)$ 代入点法式即可得到该形式。截距式在求平面与坐标轴围成的四面体体积时特别方便。

# 2. 两平面的位置关系
## 2.1 两平面的夹角
设两平面 $\Pi_1 : A_1x + B_1y + C_1z + D_1 = 0$，$\Pi_2 : A_2x + B_2y + C_2z + D_2 = 0$，它们的夹角 $\theta$（取锐角或直角）即为其法向量 $\mathbf{n}_1 = (A_1, B_1, C_1)$ 与 $\mathbf{n}_2 = (A_2, B_2, C_2)$ 的夹角：

$$
\cos\theta = \frac{|\mathbf{n}_1 \cdot \mathbf{n}_2|}{|\mathbf{n}_1|\,|\mathbf{n}_2|}
= \frac{|A_1A_2 + B_1B_2 + C_1C_2|}
{\sqrt{A_1^2 + B_1^2 + C_1^2}\,\sqrt{A_2^2 + B_2^2 + C_2^2}}
$$

注意分子加绝对值是为了保证 $\theta$ 取锐角或直角。

## 2.2 平行与垂直的判定
>[!corollary] 定理 1（两平面平行与垂直的条件）
> - **平行**（或重合）：$\displaystyle \frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2} \neq \frac{D_1}{D_2}$
>   （若比值也等于 $\dfrac{D_1}{D_2}$，则两平面**重合**）
> - **垂直**：$A_1A_2 + B_1B_2 + C_1C_2 = 0$，即 $\mathbf{n}_1 \cdot \mathbf{n}_2 = 0$
> ^plane-relation

# 3. 点到平面的距离
>[!corollary] 定理 2（点到平面的距离公式）
> 点 $P_0(x_0, y_0, z_0)$ 到平面 $Ax + By + Cz + D = 0$ 的距离为：

> $$
> d = \frac{|Ax_0 + By_0 + Cz_0 + D|}
>        {\sqrt{A^2 + B^2 + C^2}}
> $$

> ^point-plane-distance

注意到，这个公式的结构类似于平面解析几何中点到直线距离公式在三维空间的推广——分子是将点坐标代入平面方程左边取绝对值，分母是法向量的模。

>[!tip] 应用：两平行平面间的距离
> 若两平面平行（即法向量成比例），则它们之间的距离就是其中一个平面上任一点到另一个平面的距离。特别地，设 $\Pi_1: Ax+By+Cz+D_1=0$，$\Pi_2: Ax+By+Cz+D_2=0$ 平行（法向量相同），则距离为：

> $$
> d = \frac{|D_1 - D_2|}{\sqrt{A^2 + B^2 + C^2}}
> $$

# 4. 求平面方程的常见方法
## 4.1 已知三点求平面方程

设三点 $A$、$B$、$C$ 不共线，法向量可取为 $\mathbf{n} = \overrightarrow{AB} \times \overrightarrow{AC}$，再任选一点代入点法式即可。

>[!example] 例 1
> 求过点 $A(2,2,1)$，$B(0,1,3)$，$C(0,3,-2)$ 的平面方程。
>
> 解：$\overrightarrow{AB} = (-2, -1, 2)$，$\overrightarrow{AC} = (-2, 1, -3)$，

> $$
> \mathbf{n} = \overrightarrow{AB} \times \overrightarrow{AC} =
> \begin{vmatrix}
> \mathbf{i} & \mathbf{j} & \mathbf{k} \\
> -2 & -1 & 2 \\
> -2 & 1 & -3
> \end{vmatrix}
> = (1, -10, -4)
> $$

> 取 $A(2,2,1)$，由点法式得：$(x-2) - 10(y-2) - 4(z-1) = 0$，
> 即 $x - 10y - 4z + 22 = 0$。

## 4.2 已知一点和两方向向量求平面方程
若已知平面上一点和平面内的两个不共线向量（如直线的方向向量），法向量取为两方向向量的向量积。

## 4.3 角平分面

>[!definition] 定义（角平分面）
> 两相交平面的**角平分面**是到两平面距离相等的点的轨迹。由点到平面距离公式，角平分面方程为：

> $$
> \frac{|A_1x + B_1y + C_1z + D_1|}{\sqrt{A_1^2 + B_1^2 + C_1^2}}
> = \frac{|A_2x + B_2y + C_2z + D_2|}{\sqrt{A_2^2 + B_2^2 + C_2^2}}
> $$

> 去掉绝对值即得两个角平分面的方程（分别对应内外角平分面）。

# 5. 平面束

## 5.1 平面束的概念
>[!definition] 定义（平面束）
> 通过同一条空间直线 $L$ 的**所有平面**的集合，称为以 $L$ 为轴的**平面束**。

设直线 $L$ 由两平面 $\Pi_1: A_1x + B_1y + C_1z + D_1 = 0$ 和 $\Pi_2: A_2x + B_2y + C_2z + D_2 = 0$ 的交线给出。则通过 $L$ 的任意平面（除 $\Pi_2$ 本身外）的方程均可表示为：

$$
(A_1x + B_1y + C_1z + D_1) + \lambda\,(A_2x + B_2y + C_2z + D_2) = 0
$$

其中 $\lambda$ 为任意实数。写成更对称的形式（包含 $\Pi_2$）：

$$
\mu\,(A_1x + B_1y + C_1z + D_1) + \nu\,(A_2x + B_2y + C_2z + D_2) = 0
$$

其中 $\mu,\nu$ 为不全为零的实数。

注意到，平面束方程的核心思想是**用一条直线"捆绑"一族平面**——因为我们知道直线 $L$ 上的任意点同时满足 $\Pi_1$ 和 $\Pi_2$ 的方程，所以它也必然满足上述线性组合方程。这意味着我们构造的每一个平面都**自动通过直线 $L$**，无需再单独验证。

## 5.2 平面束的应用

平面束是求满足"通过给定直线 + 附加条件"的平面方程的最有力工具。一般步骤为：

1. 将给定直线写成两平面交线的形式（一般方程）；
2. 写出过该直线的平面束方程（含参数 $\lambda$）；
3. 利用附加条件（如与某平面垂直、过某点、与某平面成指定角度等）确定 $\lambda$；
4. 代回得所求平面方程。

>[!example] 例 2（通过直线和平面外一点求平面）
> 求通过直线 $L: \begin{cases} x + y + z - 1 = 0 \\ x - y + 2z = 0 \end{cases}$ 和点 $P(1, -1, 2)$ 的平面方程。
>
> 解：设过 $L$ 的平面束方程为：

> $$
> (x + y + z - 1) + \lambda(x - y + 2z) = 0
> $$

> 即 $(1+\lambda)x + (1-\lambda)y + (1+2\lambda)z - 1 = 0$。
>
> 代入 $P(1, -1, 2)$：

> $$
> (1+\lambda)\cdot 1 + (1-\lambda)\cdot(-1) + (1+2\lambda)\cdot 2 - 1 = 0
> $$

> 得 $1+\lambda - 1 + \lambda + 2 + 4\lambda - 1 = 0$，即 $6\lambda + 1 = 0$，解得 $\lambda = -\dfrac{1}{6}$。
>
> 代回得所求平面：$\dfrac{5}{6}x + \dfrac{7}{6}y + \dfrac{2}{3}z - 1 = 0$，乘以 $6$ 化简为 $5x + 7y + 4z - 6 = 0$。

>[!example] 例 3（通过直线且与已知平面垂直）
> 求通过直线 $L: \begin{cases} 2x + y - z + 1 = 0 \\ x - y + z - 2 = 0 \end{cases}$ 且与平面 $\Pi: x + 2y - z + 3 = 0$ 垂直的平面方程。
>
> 解：设过 $L$ 的平面束方程为：

> $$
> (2x + y - z + 1) + \lambda(x - y + z - 2) = 0
> $$

> 即 $(2+\lambda)x + (1-\lambda)y + (-1+\lambda)z + (1-2\lambda) = 0$。
>
> 该平面的法向量为 $\mathbf{n} = (2+\lambda,\; 1-\lambda,\; -1+\lambda)$。它与 $\Pi$ 垂直当且仅当 $\mathbf{n} \perp \mathbf{n}_\Pi = (1, 2, -1)$：

> $$
> (2+\lambda)\cdot 1 + (1-\lambda)\cdot 2 + (-1+\lambda)\cdot(-1) = 0
> $$

> 即 $2+\lambda + 2 - 2\lambda + 1 - \lambda = 0$，得 $5 - 2\lambda = 0$，$\lambda = \dfrac{5}{2}$。
>
> 代回得：$\left(2+\frac{5}{2}\right)x + \left(1-\frac{5}{2}\right)y + \left(-1+\frac{5}{2}\right)z + \left(1-5\right) = 0$，
> 化简为 $9x - 3y + 3z - 8 = 0$。

## 5.3 平面束使用注意事项
 1. 写平面束前，**必须先将直线化为两平面交线的形式**（一般方程）。如果直线以对称式或参数式给出，需先转为一般方程。
 2. 单参数形式 $(A_1x+B_1y+C_1z+D_1) + \lambda(A_2x+B_2y+C_2z+D_2)=0$ 不包含 $\Pi_2$ 本身。若题目恰好要求 $\Pi_2$，需单独检验。
3. 验证结果时，可检查所求平面是否确实通过直线上的一个点（任取直线上两点代入验证）。

# 6. 易错点总结

1. 平面方程中 $A, B, C$ 是法向量分量，**不是**平面内的方向向量——初学者常将直线的方向向量和平面的法向量混淆。
2. 截距式只适用于截距均不为零的情况；截距为零时平面过原点，不能用截距式。
3. 两平面平行时，若还要求它们重合，需额外验证 $D_1/D_2$ 是否等于法向量分量的比值。
4. 求平面夹角时，默认取**锐角或直角**（分子加绝对值）；若题意无特别说明，不应给出钝角。
5. 使用平面束时，单参数形式不包含作为基底的第二个平面本身——若答案恰好是那个平面，单参数法会"漏解"，需单独检验。
