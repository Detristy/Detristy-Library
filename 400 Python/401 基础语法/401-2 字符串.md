# 1. 访问字符串

Python使用方括号截取字符串，语法为：

```python
listName[Left:Right]
```

注意索引是从 0 开始的。

当索引只有一个的时候，仅会访问当前索引对应的字符串。

# 2. 转义字符

转义字符用于使用特殊字符。

1. `\` 在行尾处用于续行。
2. `\n` 换行。
3. `\\` 用于输出反斜杠符号。
4. `\' \"` 用于输出单引号和双引号。
5. `\b` 退格。

# 3. 字符串运算

1. `+` 字符串连接。
2. `*k` 重复输出 k 次字符串。
3. `[]` 索引访问
