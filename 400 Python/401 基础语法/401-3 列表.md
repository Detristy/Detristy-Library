# 1. 列表
列表每个值都有对应的索引，第一个位置的索引为 0 。

列表允许数据项的类型不同，例如允许：
```python
list1 = ["String",'abc',1]
```

# 2. 访问列表
## 2.1 下标索引
可以通过使用索引值来方位列表中的元素。

> [!NOTE] 反向索引
> Python支持负数索引，最后一个元素的的索引为 -1，往前一位是 -2，以此类推。

## 2.2 截取
可以使用方括号截取列表中的元素，例如：
```python
nums=[1,2,3,4,5,6]
print(nums[0:3])
```

则会输出 `[1,2,3]` 。
同样地截取也支持使用反向索引。

# 3. 列表操作
## 3.1 更新列表
可以直接通过索引对列表进行更新。

使用 `append()` 方法可以添加列表项：
```python
listName.append(obj)
```

其中 obj 是添加到列表末尾的对象。

## 3.2 删除元素
使用 `del` 来对列表中的元素进行删除：
```python
list1 = [0,1,2,3,"string1"]
del list1[0]
```

此时 list1 中只剩下 `[1,2,3,"string1"]`。
# 4. 列表方法
## 4.1 append()
在列表末尾添加新的对象。
## 4.2 count()
语法：
```python
listName.count(obj)
```

用于统计 obj 在列表中出现的次数。
## 4.3 index()
语法：
```python
listName.index(obj)
```

用于查找 obj 在列表中第一次出现的位置对应的索引。
## 4.4 remove()
语法：
```python
listName.remove(obj)
```

用于移除列表中某个值的第一个匹配项。
## 4.5 reverse()
用于反向列表中元素。
## 4.6 clear()
清空列表。
