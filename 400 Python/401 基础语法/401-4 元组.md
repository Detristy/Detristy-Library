# 1. 元组
元组和列表类似，但是不允许更改元素。

元组的声明使用小括号：
```python
tuple1 = ("string",1,2,"a")
```

> [!warning] 注意
> 只包含一个元素的时候要在元素后面添加逗号，例如：
> ```python
> tuple1 = (1,)
> ```

# 2. 元组操作
## 2.1 访问元组
元组通过下标索引进行访问，索引从 0 开始，支持反向索引。

此外元组也支持使用方括号进行截取。
## 2.2 修改元组
修改元组中的元素是非法的。

可以对元组进行连接和组合：
```python
tup1 = (12,34.5)
tup2 = ("abc","xyz")
tup3 = tup1 + tup2
print(tup3)
```
输出结果为：
```
(12,34.5,"abc","xyz")
```
## 2.3 删除元组
元组中的元素是不允许删除的，不过可以使用 `del` 对整个元组进行删除。
