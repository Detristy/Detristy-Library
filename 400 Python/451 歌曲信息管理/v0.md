#### 曲目信息
对于一首歌，其初始包含下列信息：
1. 曲名 title
2. 曲师 artist
3. 难度 difficulty ：暂时只考虑 Master 和 Re:Master
4. 等级 level
5. 定数 ds

#### 添加歌曲
在添加之前需要进行确认
```python
def add_song(songs):  
    title = input("请输入曲名：")  
    artist = input("请输入曲师：")  
    difficulty = input("请输入难度：")  
    level = input("请输入等级：")  
  
    # 保证ds输入合法  
    while True:  
        try:  
            ds = float(input("请输入定数："))  
            break  
        except ValueError:  
            print("请输入数字！")  
  
    newSong = {  
        "title": title,  
        "artist": artist,  
        "difficulty": difficulty,  
        "level": level,  
        "ds": ds  
    }  
  
    # 查看歌曲是否重复，如果重复则直接返回  
    for song in songs:  
        if song["title"] == title and song["difficulty"] == difficulty:  
            print("歌曲已存在！")  
            return  
  
    # 确定是否添加的菜单  
  
    while True:  
        choice = input("确定添加吗？(Y/N)")  
        if choice == "Y" or choice == "y":  
            songs.append(newSong)  
            save_song(songs)  
            print(f"添加成功！当前共有 {get_songs_num(songs)} 首歌曲")  
            break  
        elif choice == "N" or choice == "n":  
            print("添加失败！")  
            break  
        else:  
            print("输入错误！")
```

#### 搜索
按找曲名或曲师名模糊搜索
```cpp
def search_songs(songs): # 按曲名或曲师名搜索  
    key = input("请输入曲名/曲师名：")  
  
    found = []  
    # 查找  
    for song in songs:  
        if (  
                key.lower() in song["title"].lower()  
                or key.lower() in song["artist"].lower()  
        ):  
            found.append(song)  
    # 查找失败  
    if not found:  
        print("未找到歌曲")  
        return  
    # 查找成功  
    print(f"找到 {len(found)} 首歌曲：\n")  
    show_songs(found)
```
